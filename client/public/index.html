About CR Deck Predictor

CR Deck Predictor is a fan-made project designed to help players experiment with Clash Royale deck prediction. You can search for cards, and cards that are most played with it pops up with a percentage of how often its played with the card you selected

This site uses local data and a small JavaScript-based prediction system.

Built with plain HTML + CSS + a tiny bit of JS for the Add button.
